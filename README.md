# Sales Performance Dashboard — Excel

## Project Overview
An interactive sales dashboard built in Microsoft Excel, analysing 9,994 records 
from the Sample Superstore dataset across regions, product categories, and time periods.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## Dataset
- Source: Sample Superstore Dataset (Kaggle)
- Records: 9,994 rows
- Fields: Sales, Profit, Region, Category, Sub-Category, Order Date

## What I Analysed
- Sales and profit performance by region
- Sales breakdown by product category and sub-category
- Monthly and yearly sales trends (2014–2017)

## Key Findings
- West region generated the highest sales (~$725K) and profit
- Central region showed high sales but very low profit, suggesting excessive discounting
- Technology yields stronger profit margins than Furniture despite lower sales volume
- Sales grew over 700% between 2014 and 2017
- December consistently peaks each year, confirming seasonal demand patterns
